JavaScript

JavaScript is a language that is used to create more dynamic and interactive webpage

Operators

Operators are a symbol used to perform an operation such as arithmetic (addition, subtraction, etc.)

Variables

Variable are containers that hold other data and can be called as such.

Variable Naming Conventions

Specific style guidelines for programming. They typically cover: Naming and declaration rules for variables and functions. Rules for the use of white space, indentation, and comments.

Functions

A set of statements that performs a task or calculates a value

Methods

An action that can be performed on an object.

Arguments

A set of statements that performs a task or calculates a value

Parameters

Variables listed as a part of the function definition

Return

Stops the execution of a function and returns a value from that function.

Chaining Methods

Also known as Cascading, refers to repeatedly calling one method after another on an object, in one continuous line of code

Strings

Zero or more characters written inside quotes

Booleans

Represents one of two values: true or false.

Undefined

A global property that represents the primitive value undefined.

NaN

The isNaN() function determines whether a value is an illegal number (Not-a-Number). This function returns true if the value equates to NaN. Otherwise it returns false

Escape

A funtion that encodes a string and makes it portable, so it can be transmitted across any network.

Alert

Displays an alert box with a specified message and an OK button

Comments

Explains JavaScript code, to make it more readable.

JQuery

A concise and fast JavaScript library that can be used to simplify event handling, HTML document traversing, Ajax interactions and animation for speedy website development.

Attributes

defines how the property can be accessed
